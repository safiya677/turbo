Блог: Решение задач по шифру Цезаря

1. Определить ключи шифра Цезаря

а. АПЕЛЬСИН → САЦЬНВЩЮ

Используем русский алфавит без «Ё» (32 буквы):

А=0, Б=1, В=2, Г=3, Д=4, Е=5, Ж=6, З=7, И=8, Й=9,
К=10, Л=11, М=12, Н=13, О=14, П=15, Р=16, С=17, Т=18, У=19,
Ф=20, Х=21, Ц=22, Ч=23, Ш=24, Щ=25, Ъ=26, Ы=27, Ь=28, Э=29, Ю=30, Я=31

Проверяем сдвиг для каждой пары:

  • А(0) → С(17): (0 + k) mod 32 = 17 ⇒ k = 17
  • П(15) → А(0): (15 + 17) mod 32 = 32 mod 32 = 0 ✔
  • Е(5) → Ц(22): 5 + 17 = 22 ✔
  • Л(11) → Ь(28): 11 + 17 = 28 ✔
  • Ь(28) → Н(13): 28 + 17 = 45 mod 32 = 13 ✔
  • С(17) → В(2): 17 + 17 = 34 mod 32 = 2 ✔
  • И(8) → Щ(25): 8 + 17 = 25 ✔
  • Н(13) → Ю(30): 13 + 17 = 30 ✔

✅ Ключ: k = 17

б. АБРИКОС → ЫЬЛГЕЙМ

  • А(0) → Ы(27): k = 27
  • Б(1) + 27 = 28 → Ь ✔
  • Р(16) + 27 = 43 mod 32 = 11 → Л ✔
  • И(8) + 27 = 35 mod 32 = 3 → Г ✔
  • К(10) + 27 = 37 mod 32 = 5 → Е ✔
  • О(14) + 27 = 41 mod 32 = 9 → Й ✔
  • С(17) + 27 = 44 mod 32 = 12 → М ✔

✅ Ключ: k = 27

2. Расшифровать сообщения (0 < k < 32)

а. ФХНЗКЧ

Буквы: Ф(20), Х(21), Н(13), З(7), К(10), Ч(23)

Пробуем k = 5:

  • (20 − 5) mod 32 = 15 → П
  • (21 − 5) = 16 → Р
  • (13 − 5) = 8 → И
  • (7 − 5) = 2 → В
  • (10 − 5) = 5 → Е
  • (23 − 5) = 18 → Т

Получаем: ПРИВЕТ

✅ Расшифровка: ПРИВЕТ, ключ k = 5

б. ЦЩЕБФ

Буквы: Ц(22), Щ(25), Е(5), Б(1), Ф(20)

Пробуем k = 20:

  • (22 − 20) = 2 → В
  • (25 − 20) = 5 → Е
  • (5 − 20) = −15 mod 32 = 17 → С
  • (1 − 20) = −19 mod 32 = 13 → Н
  • (20 − 20) = 0 → А

Получаем: ВЕСНА

✅ Расшифровка: ВЕСНА, ключ k = 20

  • Е = 5 → (5 + 10) = 15 → П
  • В = 2 → (2 + 10) = 12 → М
  • А = 0 → (0 + 10) = 10 → К

✅ Зашифрованная фамилия: ЯКРЛТПМК
